### app/routes/components.py (modified, +~400 LOC)

**Helpers added:**
- `_ALLOWED_EXTS = {jpg, jpeg, png, gif, webp}`
- `_MAX_UPLOAD_BYTES = 5 MB`
- `_URL_ALLOWED_CT = {image/jpeg, image/png, image/gif, image/webp}`
- `_EXT_FROM_CT = {Content-Type -> ext}` mapping
- `_ext_from_filename(filename)` -> ext
- `_save_bytes_to_images_dir(component_id, content, ext)` -> path
  - Storage: `data/images/<id>-<uuid8>.<ext>` (UUID-named, no collision)
- `_validate_uploaded_bytes(content, ext)` -> (ok, error_msg)
  - Validates magic bytes (PNG/JPEG/GIF/WebP) not just extension

**New endpoint: `POST /api/components/{component_id}/image`**

Accepts BOTH content types:

**Multipart (`multipart/form-data`):**
- Field `image` (file)
- Optional field `source_url` (string, the original URL if pasted)
- Validates ext + magic bytes + size
- Saves to `data/images/<id>-<uuid8>.<ext>`
- Updates: `image_path`, `image_source='user_uploaded'`, `image_uploaded_at`

**JSON (`application/json`):**
- Body: `{"url": "https://..."}`
- Downloads with stdlib `urllib.request`
- Validates Content-Type from response headers
- Validates magic bytes + size
- Saves to disk
- Updates: `image_path`, `image_source='user_url'`, `image_uploaded_at`

**Response:** full updated component JSON.
**Errors:** 400 (bad ext / missing file / no URL / bad content-type), 404 (unknown component), 413 (too big), 502 (URL download failed).

**Async handler** (FastAPI 0.115's `Request.json()` requires await).
