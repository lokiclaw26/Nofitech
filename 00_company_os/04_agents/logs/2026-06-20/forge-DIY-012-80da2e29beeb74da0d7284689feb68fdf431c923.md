## What shipped (backend)

### 1. app/identity.py (NEW, 714 LOC)
- Query classifier: dev_board, sensor_module, display_module, touch_module, charger_power_module, ic_chip, passive, unknown
- Tokenizer preserves short tokens (D1, V1, R3, S3, C3, 5V, 3V3)
- Match-level: exact / variant / related / image_only / rejected
- Variant detection: token-set baseline + known-variant dict
- Ranking: identity > image, vendor > wikipedia, exact > variant

### 2. app/identity_cache.py (NEW, 222 LOC)
- SQLite verified_components + rejected_components layer
- query_signature hashing, atomic put/get

### 3. app/routes/identity.py (NEW, 128 LOC)
- POST /api/components/verify: confirm a candidate, save to verified_components
- POST /api/components/reject: record rejection, future searches skip it

### 4. app/live_search.py (modified)
- PlatformIO gate: only when component_type == dev_board
- Multi-result Adafruit/SparkFun scraping (5+ results, ranked by exact token match)
- Identity-vs-image split: search identity first, fetch images second
- Returns new fields: match_level, confidence, confidence_reason, source_quality, matched_sources

### 5. app/routes/components.py (modified)
- POST /api/components/search now returns identity-engine fields

### 6. app/main.py (modified)
- Register identity routes

### 7. app/schema.sql (modified)
- Added verified_components table with confidence + verified_by + timestamps
- Added rejected_components table with query_signature + candidate_signature + reason
- Indexes on both

### 8. tests/test_identity_engine.py (NEW, 760 LOC, 32 tests, ALL PASS)
- 7 ClassifyQueryTests (dev_boards, sensors, displays, ics, touch, ili9488_xpt2046, chargers)
- 8 IdentityEngineAcceptanceTests (one per NOFI test case)
- 4 MatchLevelTests (exact, related, image_only, variant demoted)
- 2 PlatformIOGateTests (dev_board allows, others block)
- 5 ShortTokenTests (D1, R3, S3/C3, V1, documented set)
- 3 VariantDetectionTests (d1 mini pro, no mismatch for exact, V1 rejects S3)
- 2 VerifiedCacheTests (put/get, rejected round-trip)
- 1 OldQualityBehaviorTest (regression - Stage 11 still works)
