### 2. isObstacle(c, r) helper at line 92
```js
      // Stage 11: obstacle hit-test. Linear scan of OBSTACLES; returns true
      // if any entry's col and row match (c, r). No Set, no map — 6 entries
      // makes this trivially fast.
      function isObstacle(c, r) {
        for (var i = 0; i < OBSTACLES.length; i++) {
          if (OBSTACLES[i].col === c && OBSTACLES[i].row === r) return true;
        }
        return false;
      }
```
